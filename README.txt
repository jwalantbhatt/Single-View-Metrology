Steps to be followed:
1. The input image can be changed by passing appropriate path in the starting of the code.
2. Select the 2 parallel lines in every plane.
Note: to select a parallel line we need to select 2 points on an image. This can be done by clicking on the image
at respective points.
3.Next select the origin of the image.
4. Select the x,y,z axis on the image but after every axis selection you need to enter the dimension(length) of the box.
Note: Maintain the same order in which you have selected the parallel lines ( Colors will guide you)
5. The Warp images will be directly displayed along with the projection matrix and vanishing point.
